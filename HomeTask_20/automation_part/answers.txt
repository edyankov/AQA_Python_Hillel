1. Пояснити (у файлі answers.txt) чим відрізняються Implicit and Explicit waits;

Implicit waits - це очікування, яке задається один раз на початку тесту. Це означає, що якщо елемент,
який потрібно знайти, не відображається на сторінці відразу, то Python буде очікувати певний час, поки він не з'явиться.
Якщо елемент не знайдено за певний час (який задається при налаштуванні неявного очікування), то буде викинута помилка.

Explicit waits - це очікування, яке задається в коді окремо для кожного конкретного елемента.
Вони надають більш гнучкий підхід до очікувань, оскільки дозволяють точніше контролювати таймаути та умови,
за яких має бути знайдено елемент.

2. Описати коротко (у файлі answers.txt), що таке Page Object Pattern.

Page Object Pattern - це шаблон проектування програмного забезпечення, який використовується в тестуванні веб-додатків
для розділення логіки тестів від інтерфейсу користувача. У цьому шаблоні сторінки веб-додатку розглядаються як об'єкти,
і кожна сторінка має свій відповідний клас, який містить методи та властивості для взаємодії з цією сторінкою.
Це дозволяє легко змінювати і підтримувати тести, оскільки будь-які зміни на сторінці веб-додатку відображаються
відповідним чином в класі Page Object.
